```
2. 1's Comp
Step1:(先判斷正負數)
如何判斷正負數:前面數字開頭為 1(負) 0(正)
10010010 => [1]0010010 => 負

Step2: 計算magnitude大小

Step2-1: 全部數字先執行反向(inverse)運算
10010010 => 01101101

Step2-2:算出10進位表達的數字
01101101 =>計算出10進位為 64+32+8+4+1 = 109 

Step3:整合先前的算式 => (正負號)+(大小(magnitude)) => -109
```
